¿como hacer un sandiwch?

Elige tus ingredientes:

Pan,queso,tomate,jamon,lechuga,champiñon,mayonesa,etc.



  1. Empieza colocando el pan sobre un plato.


  2. Sigue por colocar el queso.


  3. Luego sigue por colocar el tomate.


  4. Coloca despues el jamon.


  5. Para casi termniar sigue con la lechuga.


  6. Para terminar coloca el otro pan para taparlo.
